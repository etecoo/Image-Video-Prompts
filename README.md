# プロンプト生成・変換サービス

YAMLファイルからAI画像生成プロンプトを生成・変換するWebサービスです。

## 機能

- YAMLファイルのアップロードとパース
- プロンプトの生成と最適化
- 複数バリエーションの生成
- コピー機能付きのプロンプト表示
- エラーハンドリング
- レスポンシブなデザイン

## 技術スタック

### フロントエンド
- React
- js-yaml
- CSS Grid/Flexbox

### バックエンド
- Flask
- PyYAML
- Flask-CORS

## デプロイ
- Railway (自動デプロイ)
- GitHub連携
